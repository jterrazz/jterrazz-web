### Prerequisites

- Node.js 22.x.x

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/jterrazz/jterrazz-web.git
   cd jterrazz-web
   ```

2. Install dependencies:
   ```
   npm install
   ```

### Development

Run the development server:

```sh
npm run dev
```

Visit `http://localhost:3000` to view the site.
